## Stack

- Laravel
- Breeze
- MySQL
- Tailwind

## Setup

1. Clone
#####
    git clone https://github.com/AndredeQueirozPatrinicola/LaravelUserCrud.git 
#####

2. Setup .env:
####
    cd LaravelUserCrud
    cp .env.example .env
####

3. Npm
####
    npm install
    npm run dev
####

4. Run Docker

####
    docker compose up
####
